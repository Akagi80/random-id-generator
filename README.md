
# RANDOM ID GENERATOR

This is a simple random id generator.

## Instalation

`npm install @akagi80/randomid-generator`
`yarn add @akagi80/randomid-generator`

## How to use?

Import:
`const randomID = require('@akagi80/randomid-generator');`

Generate:
`console.log(randomID(10));`
